# **App Name**: CareerCompass

## Core Features:

- Job Application Table: Displays a sortable and filterable table of job applications, including columns for date applied, job title, company, status, and next steps.
- Add New Application: Allows users to add new job applications with details such as date, title, company, link and status.
- Real-time Search: Provides a search input to filter job applications by job title, company, or status in real time.
- Volunteer Role Display: Displays key details, like the volunteer role, relevant organizations, location, and URL link, about a particular volunteer role.
- AI-Powered Next Step Suggestions: Using details provided in the 'next steps' field, uses AI to suggest next steps for the user to take with their job search; the reasoning for each next step is also displayed.
- AI Cover Letter Rewrite: Using job description, uses an LLM-powered tool to help the user rewrite their cover letter. The LLM considers whether it should mention, and how much to focus on, each point in the original letter.

## Style Guidelines:

- Primary color: Indigo (#6366F1) to convey professionalism and trustworthiness.
- Background color: Light gray (#F9FAFB) to ensure readability and a clean interface.
- Accent color: Teal (#2DD4BF) to highlight important actions and statuses.
- Body and headline font: 'Inter', sans-serif, for a modern, readable, neutral feel. This font provides excellent readability and a clean aesthetic.
- Use a consistent set of minimalist icons for application statuses (e.g., applied, interviewing, offer, rejected).
- Use a card-based layout for displaying job application entries, providing a clear and organized visual structure.
- Implement subtle transition animations for adding and updating job applications to improve user experience.
